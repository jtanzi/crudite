    Welcome to the Crudite Project Page

    Project Description

    Crudite is a meal-planning application designed to solve a household need for meal planning (something I've never really liked to do). The ultimate goal is to create a web application that supports the full range of CRUD operations (create, read, update, delete) and allow one to select a set of recipes one plans to make, and generate a grocery list from that selection.

    This project is still under development and currently supports creating a recipe, viewing recipes, and deleting recipes. Features still to be added are updating recipes, more robust error handling, recipe querying and selecting, and grocery list generation.
